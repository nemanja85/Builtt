## Clone the project

```sh
git clone git@github.com:nemanja85/Builtt.git
```
```sh
cd Builtt
```

### In this project I've used Vite, TailwindCSS, React TypeScript, React Router V6, React FormHook, Easy Peasy, Yup and Playwright

## Install

```sh
yarn install
```


## Getting Started

```sh
yarn dev
```

## Run tests

```sh
yarn playwright test
```
